請描述一件產生明顯負面情緒的經歷，如何處理該情緒

在工作上除了例行性的工作外，經常會有許多臨時指派的任務或是突發事件要緊急處理，此時經常會有明顯的負面情緒出現。
而此時會把所有工作任務列出來做以下處理:

  • 哪件事讓我有負面情緒?
  • 急迫性? 重要性?
  • 是我的責任範圍嗎?
  • 我現在有能力可以處理嗎?

用以上幾個問題，釐清自己的負面情緒來源及情緒種類，確認目標並做出相對應的行動

  • 急迫性 : 緊張 => 目標: 盡速完成任務；立刻開始行動
  • 重要性 : 壓力 => 目標: 盡善盡美；盤點資源做更充足的準備
  • 非自身責任: 生氣 => 目標: 界定責任範圍 ; 協調分工
  • 非能力所及: 沮喪 => 目標: 完成任務; 尋求支援或尋找資源增加自身能力

目標確認時負面情緒都得到緩和，而後續行動則多會有正向回饋。

以下為我處理負面情緒的基礎流程:
察覺起因 => 釐清情緒種類 => 設立目標 => 緩和負面情緒 => 開始行動 => 正向情緒回饋
