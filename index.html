請描述一件產生明顯負面情緒的經歷，如何處理該情緒

在工作上除了例行性的工作外，經常會有許多臨時指派的任務或是突發事件要緊急處理，此時經常會有明顯的負面情緒出現。
而此時會把所有工作任務列出來做以下處理：

  • 哪件事讓我有負面情緒?
  • 急迫性? 重要性?
  • 是我的責任範圍嗎?
  • 我現在有能力可以處理嗎?

用以上幾個問題，釐清自己的負面情緒來源及情緒種類，確認目標並做出相對應的行動，舉例來說：

  • 事件：處理繁複的例行工作時，20分鐘後有一個會議，突然有緊急事件要處理
  • 情緒來源：會議可能會遲到讓我有情緒，情緒為緊張與焦慮
  • 急迫性： 緊急事件 > 會議 > 例行事務
  • 重要性：會議 > 緊急事件 > 例行事務
  • 責任範圍：例行工作：本職業務； 緊急事件： 支援業務; 會議：本職業務；
  • 處理能力：例行工作：隨時可以處理但需要時間； 緊急事件：短時間內只有我處理但所需時間不明；會議：必須參與且沒有調動時間的權限
  • 確認目標：準時進入會議
  • 行動：延後例行工作並設定提醒後，立刻處理緊急事件；處理時評估所需時間，若需時超過20分鐘，則請求支援，盡速在會議前處理完。

在釐清並確認目標的過程，負面情緒會逐步緩和，而後續行動則會有正向回饋。

以下為我處理負面情緒的基礎流程：
1. 察覺起因 => 2. 釐清情緒種類 => 3.確立目標 => 4. 緩和負面情緒 => 5. 開始行動 => 6.正向情緒回饋

在工作中我不斷練習，步驟已簡化成1. 察覺起因 => 3.確立目標=> 5. 開始行動
如果遇到特殊情形，如察覺不到起因，則會從眼前能著力的事件開始處理，現在工作上大多平靜愉快。

  • 參考chrome developTool提供ipad pro的寬度，設定flex直向排版與橫向排版切換的斷點在1024px。
